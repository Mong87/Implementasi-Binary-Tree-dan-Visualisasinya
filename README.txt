TREE (PYTHON)

Cara menjalankan:
1. install graphviz: "pip install graphviz" di terminal dan software https://graphviz.org/download/
   pip install graphviz

2. jalankan program:
   python main.py

Fitur:
- insert data ke binary search tree
- traversal (preorder, inorder, postorder)
- visualisasi tree (teks & diagram/graph)
